# hakkoT12-arduino-hardware
Hardware part of simple HAKKO T12 soldering station based on this firmware https://github.com/sfrwmaker/hakko_t12

Drawed in Cadsoft Eagle 7.7.0

Single sided pcb, only THT parts.
